# Facial Emotion Recognition using HOG and SVM

A machine learning project investigating facial emotion recognition using traditional computer vision techniques. The system combines Histogram of Oriented Gradients (HOG) feature extraction with Support Vector Machine (SVM) classification to identify facial expressions from image datasets.

## Technologies Used

- Python
- OpenCV
- scikit-learn
- NumPy
- Matplotlib
- Google Colab

## Features

- Face detection and preprocessing
- HOG feature extraction
- SVM classification
- Emotion prediction
- Performance evaluation
- Confusion matrix analysis

## Datasets

- CK+ (Extended Cohn-Kanade Dataset)
- JAFFE (Japanese Female Facial Expression Dataset)

## Project Objectives

- Investigate facial emotion recognition using traditional machine learning techniques
- Compare performance across multiple facial expression datasets
- Evaluate classification accuracy and model robustness
- Analyse strengths and weaknesses of handcrafted feature approaches

## Methodology

1. Detect and preprocess facial images
2. Extract HOG (Histogram of Oriented Gradients) features
3. Train Support Vector Machine (SVM) classifiers
4. Evaluate using accuracy, precision, recall and F1-score
5. Compare performance across datasets

## Results

- Achieved stronger performance on the JAFFE dataset due to its controlled conditions
- Identified challenges in distinguishing visually similar emotions
- Established a traditional machine learning baseline for comparison with deep learning approaches

## Key Learning Outcomes

- Computer vision fundamentals
- Feature engineering using HOG descriptors
- SVM classification techniques
- Model evaluation and performance analysis
- Facial emotion recognition workflows

## Future Improvements

- Replace handcrafted features with deep learning approaches
- Implement Convolutional Neural Networks (CNNs)
